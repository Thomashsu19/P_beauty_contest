現在共同參與這個遊戲的共有 {{ group.player_num }} 位玩家！
遊戲規則如下：
本遊戲共有 {{ C.NUM_ROUNDS }} 回合。
每一回合中，每個玩家都要從最小為 0，最大為 100 的區間裡，輸入一個整數。
所有玩家都送出輸入的整數後，電腦將計算所有玩家輸入的「平均數」。
電腦會將這個「平均數」， {{ if group.is_twothird }} 乘以2⁄3 倍作為「 勝利數字 」。
{{ else }} 乘以2⁄3 倍作為「 勝利數字 」。
{{ endif }}
若您挑選的整數，在所有{{ group.player_num }}位玩家中最接近「 勝利數字 」，您就是該回合的贏家。

在每回合的結果頁面，每位玩家都可以看到該回合所有玩家的平均數字、勝利數字、和贏家所選擇的數字及當回合所獲得的報酬。
所有人都確認完結果後，將進到下一回合的遊戲。
請注意，每回合的選擇都是獨立的，您在一回合的選擇，不會影響您在下一回合的輸贏機率。 每回合的贏家可獲得100元新台幣的獎金，若該回合贏家大於一人，系統將在所有贏家中，隨機選擇一人發放獎金。 在遊戲結束後，系統會告知您在本場實驗中的總報酬。
