{{ block title }} 實驗說明 {{ endblock }} {{ block content }}

歡迎，感謝您參與今日的實驗！

待會您將與另外 {{ num_player_1 }} 位玩家，玩 {{ if group.is_twothird }} 「 2⁄3 倍的平均數」的遊戲。 {{ else }} 「 1⁄2 倍的平均數」的遊戲。 {{ endif }}

本遊戲共有 {{ C.NUM_ROUNDS }} 回合。

每一回合中，所有玩家皆需要從 {{ C.timeout_sec }} 秒內，從最小為 0，最大為 100 的區間裡，挑選一個整數。
挑選完畢後，系統將會計算所有已提交數字之平均數，並將平均數乘以 {{ if group.is_twothird }} 2⁄3 倍作為「 勝利數字 」。
{{ else }} 2⁄3 倍作為「 勝利數字 」。
{{ endif }} 若您挑選的整數，在所有玩家中最接近「 勝利數字 」，您就是該回合的贏家，將獲得報酬 {{ C.winning_prize }} 元新台幣。
惟須注意，當最接近「 勝利數字 」的玩家有兩位以上時，該回合的所有贏家將均分報酬 {{ C.winning_prize }} 元新台幣。
若您不是贏家或未在時間內做出決策，則獲得報酬 {{ C.consolation_prize}} 元新台幣。

若未於時限內提交數字，系統將會視之為空值，不計入當回合平均數之計算。
在每回合的結果頁面，每位參與者都可以看到當回合贏家所選擇的數字及當回合所獲得的報酬。
結果頁面將顯示 {{ C.timeout_sec_result }} 秒。
請記住，每回合選擇之間都是獨立的。
在遊戲結束後，系統會結算您在本場實驗中的總報酬。

例如：
以下是您與其他 2 名玩家在某一回合中選擇的數字。
您選擇的數字：56
其他 2 名玩家選擇的數字：7 99
{{ if group.is_twothird }} 在這個回合當中，平均數為 54，勝利數字為 36 。
由於 56 最接近勝利數字 36，您是本回合的贏家，可獲得報酬 {{ C.winning_prize }} 元新台幣。
其他選擇 7 與 99 的玩家，可獲得報酬 {{ C.consolation_prize }} 元新台幣。
{{ else }} 在這個回合當中，平均數為 54，勝利數字為 27 。
由於 7 最接近勝利數字 27，您是本回合的贏家，可獲得報酬 {{ C.winning_prize }} 元新台幣。
其他選擇 56 與 99 的玩家，可獲得報酬 {{ C.consolation_prize }} 元新台幣。
{{ endif }}
